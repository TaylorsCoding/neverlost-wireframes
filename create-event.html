Create Event Page

Form exists here to create an event.

Includes:

  • Title (required)
  • type of event (required) (dropdown)
  • zip code (strongly recommended)
  • date published(automatic, not included in form)
  • animal if event is about an animal (dropdown that includes new animal)

    • if a new animal is put in here, then the following section is included:

THIS IS FOR A NON PET ANIMAL

Form exists here to create an animal who may not have an owner.

Includes:

  • name
  • gender
  • species/breed
  • description
  • upload picture
The upload picture will create an entry in the picture table in the database that references the pet that is being inserted into the animal table.

Form exists here to create a pet.

Includes:

  • name
  • gender
  • species/breed
  • description
  • upload picture
The upload picture will create an entry in the picture table in the database that references the pet that is being inserted into the animal table.
