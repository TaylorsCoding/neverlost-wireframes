Create Organization Page

Form exists here to create an organization.

Includes:

  • Name (required)
  • website (required)
  • phone number (required)
  • zip code (required)

If they press submit, and they fill in all the required information, their organization model is created and they are taken to their organization page

If the user doesn't submit some answer for all required fields, messages will display informing the user that they need to.

This organization will not belong to the user. It will remain not owned by any user
