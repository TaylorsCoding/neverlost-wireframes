Create Account Page

Form exists here to create an account.

Includes:

  • username (required)
  • email (required)
  • first and last name (not required)
  • zip code (not required but recommended, and will explain why recommended)
  • password and confirm password (required)
  • a checkbox asking if they want to be able to be seen by other users (required) (some information will be provided about what this means)

If they press submit, and they fill in all the required information, their user model is created and they are taken to their user page

If the user doesn't submit some answer for all required fields, messages will display informing the user that they need to.
